## 1. Popis projektu
Textový analyzátor v Pythonu, který po úspěšném přihlášení uživatele umožňuje analyzovat vybraný text z předem připravené sady textů. Program provádí základní statistickou analýzu textu a zobrazuje jednoduchý textový sloupcový graf četnosti délek slov.

---

## 2. Funkcionalita
- Přihlášení uživatele pomocí jména a hesla   
- Výběr textu pomocí číselného označení  
  - neplatný nebo nečíselný vstup ukončí program  
- Analýza textu:
  - počet slov  
  - počet slov začínajících velkým písmenem  
  - počet slov psaných **VELKÝMI** písmeny  
  - počet slov psaných malými písmeny  
  - počet čísel a jejich součet  
- Textový sloupcový graf znázorňující četnost délek slov

---

## 3. Registrovaní uživatelé
Pro přihlášení do programu použijte některou z následujících dvojic jméno–heslo:

| Uživatelské jméno | Heslo        |
|------------------|-------------|
| bob              | 123         |
| ann              | pass123     |
| mike             | password123 |
| liz              | pass123     |

---

## 4. Spuštění programu
1. Spusťte program  
2. Zadejte uživatelské jméno a heslo (viz sekce **Registrovaní uživatelé**).  
3. Vyberte text podle čísla.  
4. Program zobrazí výsledky analýzy a graf četnosti délek slov.
